# Guided Mini Website Simulator (HTML & CSS)

## Description

This repository contains an educational simulator designed to support the creation of simple multi-page websites using HTML and CSS.

The project is intended for students in the early stages of learning web development. The environment guides learners through the design and implementation of a mini website focused on a selected country, structured across multiple interconnected pages.

The simulator can be used online via a web browser or locally by opening the `index.html` file.

---

## Educational Objectives

- Understand the basic structure of an HTML page
- Use headings, paragraphs, lists, links, and images
- Organize a multi-page website
- Apply simple embedded CSS styling
- Manage file names and relative paths correctly
- Develop original content while avoiding passive copy-paste
- Produce a coherent and functional digital artifact

---

## Features

- Guided content creation
- Automatic HTML code generation
- Integrated preview system
- Multi-page website structure
- Basic validation of requirements
- Download of generated pages
- Example of a complete high-quality project
