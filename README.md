# Simulatore didattico per mini siti web in HTML e CSS

## Descrizione

Questo repository contiene un simulatore didattico per la progettazione e la 
realizzazione di mini siti web statici in HTML e CSS.

Il progetto è pensato per studenti in fase iniziale di apprendimento dello sviluppo web.
L'ambiente guida lo studente nella costruzione di un mini sito dedicato a uno Stato del mondo, articolato in più pagine collegate tra loro.

Il simulatore è utilizzabile online tramite Netlify oppure localmente aprendo il file 
'index.html' nel browser.

---

## Obiettivi didattici

- Comprendere la struttura di base di una pagina HTML.
- Usare titoli, paragrafi, liste, link e immagini.
- Organizzare un sito multipagina.
- Usare CSS incorporato in modo semplice.
- Gestire nomi file e percorsi relativi.
- Rielaborare contenuti evitando il copia/incolla passivo.
- Consegnare un prodotto digitale semplice, coerente e navigabile.
